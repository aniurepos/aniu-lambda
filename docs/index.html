Features

Why developers choose ChatQuota

Built for people who are tired of clicking through the AWS Console just to answer a simple question.

Ask in plain English

Stop digging through the AWS Console. Type what you want — 'How many Lambda invocations this month?' — and get an instant, accurate answer.

Zero setup for your team

Deploy once with CDK, share the API key. Everyone on your team checks usage without touching AWS IAM or CloudWatch.

Your data stays yours

API Key auth via Usage Plans. Credentials live in your browser — nothing is stored on our servers. Ever.

Real-time CloudWatch APIs

Queries hit your own AWS account directly through API Gateway. No middleman. No data pipelines. Just fast answers.

Built for teams

Unlimited users share a single deployment. Onboard new engineers in seconds — send them the URL and API key.

Open source & extensible

The full stack is open source. Customize the CDK, add new query types, or contribute back — it's all on GitHub.
How it works

Get started in 3 minutes

One command to deploy. One URL to connect. Then just ask.
1

Deploy the CDK stack

Run `cdk deploy` in your AWS account. This provisions a Lambda function, API Gateway, and API Key — all in under 3 minutes.
2

Open the app & connect

Launch the web app or download the macOS desktop app. Paste your API Gateway URL and API Key — no IAM roles to configure.
3

Ask questions. Get answers.

Type natural language questions about your AWS usage. ChatQuota translates them into API calls and returns clear, actionable answers.
